2048 AI
========

2048 인공지능

### Installation

1.  Clone the sources from git

```
git init https://github.com/simnalamburt/2048-ai.git
```

1.  Build & Play

```
cd 2048-ai
make
open index.html
```

### Requirements

* make
* [node.js](http://nodejs.org/)
* [stylus](http://learnboost.github.io/stylus/)
* [LiveScript](http://livescript.net/)

### Dependencies

* Normalize.css
* jQuery
